## 3. How far can we go?
With MDX, the line between "content" and "app" disappears. You can:
1. **Scrollytelling**: Trigger animations as the student reads.
2. **Live Code Editing**: Embed a code editor that executes live (like we do with WASM).
3. **Data Visualization**: Render charts that update based on user input elsewhere in the lesson.

For example, inline math also works: <KatexMath expr="a^2 + b^2 = c^2" />
